# Accounting Ledger
Terminal app built in Java that helps you keep track of your finances by recording them inside a .csv file.

---

## About
Accounting Ledger is a Java terminal app designed to make personal finance tracking simple.
It lets you log deposits and payments, browse your transaction history, and search through
your records, all stored locally inside a .csv file.


## Features
- **Deposits:** Record credit transactions with a date, amount, description, and vendor
- **Payments:** Record debit transactions with a date, amount, description, and vendor
- **Ledger View:** Browse all transactions, deposits only, or payments only
- **Reports:** Search with built-in custom search filters

## Design Process
- I wrote down what I wanted to start coding first along with what repos I've worked on in the past I will probably need to refer to when working on this app.
- I knew I wanted to do the homescreen for the app and have the log working, so I focused on those 2 first.
- After that I finished most of the screen and finished the log along with creating the arraylist that the app will be using.
- All that was left was the reports menu and custom search which both reused a lot of code outside of LocalDate usage.
